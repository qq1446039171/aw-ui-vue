# Card
卡片组件

### 示例
<aw-card imgSrc="/logo.png" summary="Vue"/>

### 代码
```html
<aw-card imgSrc="/logo.png" summary="Vue"/>
```

### Attributes
| 属性 | 说明 | 类型 | 是否必要 | 默认值 |
| :--- | :--- | :--- | :--- | :--- |
| width | 卡片的宽度 | Number | false | - |
| imgSrc | 图片资源地址 | String | true | - |
| imgHeight | 图片高度 | Number | false | - |
| summary | 卡片的概要 | String/Slot | false | - |
| footer | 卡片底部 | Slot | - | - |
